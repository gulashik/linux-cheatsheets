Интерактивная шпаргалка команды find для Linux

конструктор команды
📍 Где искать
. текущая /home/user /var/log ~ домашняя / корень
🔤 Имя файла
-name "*.log" -name "*.py" -iname "*.TXT" -name "config*" -name "*.sh"
📁 Тип
-type f файл -type d директория -type l симлинк -type p pipe
🕐 Время
-mtime -7 изм <7д -mtime +30 изм >30д -atime -1 доступ <1д -newer file -mmin -60 <60мин
⚖️ Размер
-size +100M -size -1k -size +1G -empty пустые
🔒 Права
-perm 777 -perm /u+x exec -perm -4000 SUID -user root -group www-data
🔢 Глубина
-maxdepth 1 -maxdepth 3 -mindepth 2
⚙️ Действие
-print -ls -delete ⚠️ -exec ls -lh {} -exec rm {} ⚠️ xargs grep
🔗 Логика
-not -o OR \( … -o … \)
предпросмотр команды
find .
примеры — нажми чтобы скопировать
удалить старые логи
find . -name "*.log" -mtime +30 -delete
найти SUID файлы (безопасность)
find / -perm -4000 -type f 2>/dev/null
большие файлы с деталями
find . -type f -size +50M -exec ls -lh {} \;
найти TODO в Python файлах
find . -type f -name '*.py' | xargs grep -l 'TODO'
изменённые за сутки
find /home -mtime -1 -type f -ls
удалить пустые папки
find . -type d -empty -delete
исключить node_modules
find . -not -path "*/node_modules/*" -name "*.js"
очистить архивные логи
find /var/log -name "*.gz" -mtime +7 -exec rm {} \;
